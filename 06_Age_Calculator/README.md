# Age Calculator

A simple web application that calculates your age based on your date of birth.

## Features

- **Date Input**: Select your date of birth using an HTML date picker
- **Age Calculation**: Calculates your exact age in years, accounting for months and days
- **User-Friendly Interface**: Clean and simple UI for easy interaction

## How to Use

1. Open `index.html` in a web browser
2. Select your date of birth from the date input field
3. Click the "Calculate Age" button
4. Your age will be displayed on the screen

## Files

- `index.html` - Main HTML structure
- `main.js` - JavaScript logic for age calculation
- `style.css` - Styling for the application
- `README.md` - Project documentation

## Technical Details

The age calculation considers:
- The difference in years
- The difference in months
- The current day of the month

This ensures accurate age calculation even if your birthday hasn't occurred yet in the current year.

## Browser Compatibility

Works in all modern browsers that support:
- HTML5 date input
- ES6 JavaScript
