# Random Password Generator

A simple and interactive web application that generates secure random passwords with a single click.

## Features

- **Generate Random Passwords**: Click the "Generate" button to create a 14-character random password
- **Copy to Clipboard**: Click the copy icon to instantly copy the generated password to your clipboard
- **Password Confirmation**: Visual feedback confirming when a password has been copied
- **Responsive Design**: Clean, centered layout with a modern pink-themed UI
- **Special Characters**: Passwords include uppercase letters, lowercase letters, numbers, and special characters for enhanced security

## Password Composition

Each generated password contains 14 random characters from the following set:
- Numbers: `0-9`
- Lowercase letters: `a-z`
- Uppercase letters: `A-Z`
- Special characters: `!@#$%^&*()_+?:{}[]`

## How to Use

1. Open `index.html` in your web browser
2. Click the **Generate** button to create a new random password
3. The generated password will appear in the input field
4. Click the **copy icon** next to the password to copy it to your clipboard
5. A notification will appear confirming the password has been copied
